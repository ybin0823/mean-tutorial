# MEAN stack tutorial
**What is MEAN ?**
MongoDB + Express + AngularJs + Node.js
Back-end쪽 DB는 NoSQL의 한 종류인 MongoDB를 사용하고 웹 서버는 node.js의 express를 사용한다. 그 외 서버 쪽에서 필요한 기능은 node.js의 module을 설치한다. Front-end쪽은 AngularJs로 작업한다.
